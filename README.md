## Publish commands:

```
Angular:

npm install -g angular-cli-ghpages
ng build --prod --base-href "https://kiranbangale.github.io/"
ngh --message="Build release" --branch="portfolio"
```

```
Firebase:

npm i -g firebase-tools
firebase init
firebase serve --only functions,hosting
firebase deploy
```
